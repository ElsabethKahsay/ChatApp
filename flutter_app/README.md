## Features

### Cute Features

- **Pet Mood Indicator**: A little mascot reacts to the number of unread messages. The mascot's mood changes dynamically based on the count of unread messages:
  - Happy: No unread messages.
  - Curious: A few unread messages.
  - Overwhelmed: Many unread messages.

### Security Note

Don’t reuse long-lived shared secrets forever. Add a ratchet/rotation later.
